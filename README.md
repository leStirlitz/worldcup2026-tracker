# 🏆 FIFA World Cup 2026 Tracker

Live knockout bracket and player statistics for the 2026 FIFA World Cup.

## Features

- **Knockout Bracket** - Visual bracket from Round of 32 to Final
- **Player Stats** - Golden Boot race, top assist providers, star player tracker
- **Match Schedule** - Upcoming matches with CET timezone
- **Highlights Links** - Quick access to Sporza YouTube highlights

## Live Site

🌐 **[View Live Bracket](https://lestirlitz.github.io/worldcup2026-tracker/)**

## Auto-Updates

This site automatically updates twice daily via GitHub Actions:
- 🌅 6:00 AM CET (Morning update)
- 🌙 Midnight CET (Evening update)

You can also trigger a manual update from the Actions tab.

## Local Development

Just open `index.html` in your browser - no build process needed!

## Data Sources

- Match results from official FIFA sources
- Player statistics from ESPN, Fox Sports, Sporting News
- Highlights from [Sporza YouTube](https://www.youtube.com/@Sporza_be)
